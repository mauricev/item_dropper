# Item Dropper Example

A simple example demonstrating the basic usage of the `item_dropper` package.

## Features Demonstrated

- **Single-Select Dropdown**: Choose your favorite fruit from a list
- **Multi-Select Dropdown**: Select multiple colors with pre-selected items and a max limit of 4

## Running the Example

```bash
flutter run
```

## What This Shows

This minimal example demonstrates:

- Basic setup for both widget types
- How to handle selection changes
- Pre-selecting items in multi-select
- Setting a maximum selection limit
- Displaying selected items

For more advanced features (custom styling, add/delete items, group headers, etc.),
see the [main package documentation](https://pub.dev/packages/item_dropper).
